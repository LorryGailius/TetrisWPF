# TetrisWPF
A tetris game made with C# .NET WPF

## How to play
| Key       | Action              |
| --------- | ------------------- |
| ↑ / W     | Rotate Clockwise    |
| → / D     | Move block left     |
| ← / A     | Move block right    |
| ↓ / S     | Move block down     |
| Space     | Drop block          |
| Shift     | Hold block          |
